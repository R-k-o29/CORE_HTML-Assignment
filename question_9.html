Common formatting tags

  1. < b > - Bold tag
    • Used to write Bold letters
    • Example : BOLD
  2. < i > - Italic tag
    • Used to write texts in italic
    • Example : Italic
  3. < del > - Delete tag
    • Used to delete a text
    • Example : Delete this text
  4. < sub >- Subscript text
    • Used to write text in Subscript
    • Example : H 2O
